# Классическое машинное обучение: Практические задачи

Репозиторий содержит решения трех практических задач по классическому машинному обучению, включая анализ данных, предварительную обработку, визуализацию и построение предиктивных моделей.

## Структура проекта

### **Задача 1: Обнаружение фальшивых новостей**
- **Стек:** `pandas`, `sklearn`, `plotly`, `seaborn`
- **Описание:** Модель бинарной классификации для определения достоверности новостей (`REAL`/`FAKE`). В качестве векторизатора текста использован `TfidfVectorizer`, а алгоритмом классификации выступил онлайн-алгоритм `PassiveAggressiveClassifier`.

### **Задача 2: Обнаружение болезни Паркинсона**
- **Стек:** `xgboost`, `sklearn`
- **Описание:** Выявление болезни Паркинсона на ранних стадиях с использованием экстремального градиентного бустинга (`XGBoost`). Включает масштабирование признаков и анализ важности признаков (`Feature Importance`).

### **Задача 3: Предсказание оттока клиентов (Customer Churn) — Творческое задание**
- **Стек:** `pandas`, `sklearn`, `xgboost`, `matplotlib`, `seaborn`
- **Описание:** Полноценный пайплайн анализа оттока клиентов телеком-компании, реализованный в ООП-парадигме. Сравниваются 4 алгоритма (`XGBoost`, `Random Forest`, `SVM`, `Logistic Regression`). Реализована автоматическая визуализация `ROC-AUC`, `Precision-Recall` кривых и матриц ошибок.
